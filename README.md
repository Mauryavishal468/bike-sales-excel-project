# 🚲 Bike Sales Excel Project

An Excel-based analysis of bike purchase data based on customer demographics.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Basic Data Cleaning

## 📈 Data Fields
- Age, Gender, Region
- Income, Marital Status, Children
- Purchased Bike (Yes/No)

## 📁 Files Included
- `Bike Sales Excel Project.xlsx`
